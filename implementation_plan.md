# Implementation Plan

[Overview]
Реализация всех оставшихся секций главной страницы интернет-магазина мебели CLICKWOOD по предоставленному дизайну.

Цель - полностью закончить верстку главной страницы, сохранив существующий стиль, типографику и цветовую палитру проекта, используя предоставленный макет как визуальный референс. Все секции будут реализованы в едином эстетическом стиле с соблюдением отступов, пропорций и визуальной иерархии.

[Types]
Изменения в типах не требуются, используется существующая конфигурация Tailwind CSS.

Используемые стили и константы:
- Цвета: primary #788678, primaryDark #667566, textMain #523E33
- Шрифты: heading - Roscha, body - Arsenal SC
- Размеры: стандартные контейнеры, отступы согласно существующему коду
- Адаптивность: мобильная, планшетная и десктопная версия

[Files]
Будет модифицирован один существующий файл, новые файлы не создаются.

Изменения:
- pages/index.vue - добавление всех отсутствующих секций после Hero блока
  ✅ Секция подписки с акционной скидкой
  ✅ Секция "Новая коллекция" с карточками товаров
  ✅ Секция "О нас" с текстом и изображениями
  ✅ Секция "Наши работы" с примерами интерьеров
  ✅ Подвал сайта с логотипом

[Functions]
Новые функции не требуются, вся реализация выполняется средствами шаблона Vue и Tailwind CSS.

Добавляются только статичные компоненты верстки, интерактивность будет добавлена на следующих этапах.

[Classes]
Новые классы не создаются, используются существующие утилиты Tailwind CSS согласно конфигурации проекта.

[Dependencies]
Новые зависимости не требуются, все используемые пакеты уже установлены в проекте.

[Testing]
После реализации необходимо проверить:
✅ Отображение всех секций в правильном порядке
✅ Соответствие цветовой палитры и типографики
✅ Адаптивность на разных разрешениях экрана
✅ Отсутствие горизонтального скролла
✅ Корректное отображение изображений и отступов

[Implementation Order]
Последовательность реализации секций:

1. Добавить секцию подписки с зеленым фоном, полем ввода email и кнопкой подписки
2. Добавить секцию "Новая коллекция" с заголовком и 3 карточками товаров
3. Добавить секцию "О нас" с текстовым описанием и сеткой изображений
4. Добавить секцию "Наши работы" с примерами реализованных интерьеров
5. Добавить подвал сайта с логотипом и зеленым фоном
6. Настроить отступы между секциями и общую композицию страницы
7. Проверить адаптивность и исправить визуальные несоответствия
