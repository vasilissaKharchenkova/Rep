# Implementation Plan

[Overview]
Исправление всех неисправностей Swiper слайдера: неработающая кнопка вперёд, отсутствие плавности, дёрганое переключение слайдов и неработающий свайп курсором с зажатой ЛКМ.

Проблемы возникают из-за неправильной конфигурации Swiper 12 в Nuxt 4, отсутствия нужных модулей, некорректных параметров инициализации и отсутствия модуля Manipulation который необходим для корректной работы loop режима.

[Types]  
Изменения не требуют добавления новых типов, только корректировка параметров инициализации Swiper.

Необходимо добавить недостающий модуль `Manipulation`, исправить параметры скорости, сопротивления, переходов и добавить оптимальные настройки для плавной анимации.

[Files]
Будет модифицирован только один существующий файл.

Детальный перечень изменений:
- ✅ `Rep/pages/index.vue` - корректировка импортов, параметров инициализации свайперов
- ❌ Новые файлы не требуются
- ❌ Файлы для удаления отсутствуют
- ❌ Конфигурационные файлы не изменяются

[Functions]
Существующие функции остаются без изменений, добавляются только правильные параметры конфигурации.

Детальный перечень изменений:
- Импортировать модуль `Manipulation` из Swiper
- Добавить `simulateTouch: true` и `touchRatio: 1`
- Исправить `speed` на 600ms с плавной кривой
- Добавить `ease: 'ease-out'` для анимации
- Отключить `preventInteractionOnTransition`
- Добавить `longSwipes: true`, `shortSwipes: true`
- Исправить параметры loop режима
- Добавить `touchStartPreventDefault: false` для корректной работы перетаскивания

[Classes]
Новые классы не создаются, корректируется только конфигурация экземпляров Swiper.

Детальный перечень изменений:
- Для обоих свайперов (`collection-slider` и `works-slider`) применяются идентичные исправления конфигурации
- Добавляются все необходимые параметры для корректной работы перетаскивания мышью
- Исправляются параметры отвечающие за плавность анимации

[Dependencies]
Зависимости не требуют изменений, используется текущая версия Swiper 12.1.3.

Все необходимые модули уже присутствуют в установленном пакете, требуется только их правильный импорт.

[Testing]
После исправлений необходимо проверить:
1. Работа кнопок вперёд и назад без задержек
2. Плавность переключения слайдов
3. Возможность перетаскивать слайд мышью с зажатой левой кнопкой
4. Работа бесконечного режима loop
5. Отсутствие дёрганий при быстром переключении
6. Корректная работа на мобильных устройствах

[Implementation Order]
Исправления реализуются последовательно по одному файлу.

1. Добавить импорт недостающего модуля Manipulation из Swiper
2. Добавить Manipulation в список modules для обоих свайперов
3. Исправить параметр speed и добавить easing функцию
4. Добавить параметры отвечающие за перетаскивание мышью
5. Исправить параметры loop режима
6. Добавить оптимальные настройки для плавности анимации
7. Проверить и убрать лишние параметры которые создают конфликты
